  3. Clear button: Clear the screen and start over.
  4. Line width: Use these buttons to change the width of the drawing pen/line.
  5. Entrance Door: Your game character will spawn here. Make sure to draw something to catch him.
  6. Exit Door: Reach this door to win the game.
  7. Red: Draw with this to create dangerous objects.
  8. Black: Draw with this to create platforms (objects that don't move).
  9. Pale blue: Draw with this to create very bouncy platforms - high elasticity.
  10. Blue: Draw with this to create bouncy platforms - medium elasticity.
  11. Dark blue: Draw with this to create slightly bouncy platforms - low elasticity.
  12. Play Button: Click this to play your game.
